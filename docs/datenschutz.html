2. Welche Daten werden verarbeitet?

MyRehaGuide verarbeitet keine personenbezogenen Daten auf externen Servern. Alle Daten (Favoriten, Einstellungen, gewählte Version) werden ausschließlich lokal auf Ihrem Gerät gespeichert.

  • Favorisierte Übungen: lokal gespeichert (AsyncStorage)
  • Farbschema-Einstellung: lokal gespeichert
  • App-Version (kostenlos/Vollversion): lokal gespeichert

3. Keine Weitergabe an Dritte

Es werden keine Daten an Dritte weitergegeben. Die App benötigt keine Internetverbindung für die Kernfunktionen. Es werden keine Analysetools, Tracker oder Werbedienste eingesetzt.

4. Seite „Signal for Help"

Die Seite „Signal for Help" (Prävention) speichert bewusst keine Daten und ist ohne Login zugänglich – damit auch Personen in kontrollierten Umgebungen diese Information lesen können.

5. Notrufnummern

Wenn Sie in der App eine Notrufnummer antippen, wird Ihr Telefon die Nummer wählen. Dabei können je nach Mobilfunkanbieter Verbindungsdaten anfallen. Wir haben keinen Zugriff auf diese Daten.

6. Ihre Rechte

Da keine personenbezogenen Daten auf unseren Servern gespeichert werden, können Daten jederzeit durch Deinstallation der App vollständig gelöscht werden. Bei Fragen wenden Sie sich an: heiess@icloud.com
